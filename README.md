# Student Information System

A simple console-based Student Information System written in Python.

This application allows users to manage student records by adding, deleting, listing students, assigning grades, and calculating the class average. Student data is stored persistently using a text file.

---

- Add and delete students  
- List all students

- View individual student information  
- Calculate average grade  
- Persistent file-based storage

## How to Run

1. Make sure Python 3 is installed on your system.

2. Clone the repository:
   ```bash
   git clone https://github.com/USERNAME/student-information-system.git
   ```
   
3. Navigate to the project directory:
 ```bash
 cd student-information-system
 ```

4. Run the application:
```bash
python main.py
 ```
